This tutorial starts where the Quick Start left off. All the code that is required however is in this tutorial so you don't have to have finished that tutorial to do this one. If you'd like to start here you are able to without reference to the other tutorial.

Setup from Scratch

  • Install Maven 2 (http://maven.apache.org)
  • Get the code from http://svn.apache.org/repos/asf/cayenne/main/trunk/docs/quick-start-rop/ with svn, i.e.
    • svn co http://svn.apache.org/repos/asf/cayenne/main/trunk/docs/quick-start-rop/
  • Build Eclipse config files (or your favorite IDE)
    • cd quick-start-rop
    • mvn eclipse:eclipse
      • Eclipse Info
      • IDEA Info
      • NetBeans Info
  • Start CayenneModeler and add derby.jar to the modeler CLASSPATH. Open "cayenne-rop-server-tutorial/src/main/resources/cayenne.xml" in CayenneModeler and generate a database schema (skip the Java class generation part, just do the database).
    • You might also want to change the path to the db, its currently set to /tmp/testdb, which works fine on *nix but not so nicely on Windows.
  • From the cayenne-rop-server-tutorial directory start jetty with the cayenne bits deployed
    • mvn jetty:run-war
  • You are now running the Cayenne ROP server app
