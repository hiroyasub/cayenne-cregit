Introduction

As of version 3.0 Cayenne includes a Java Persistence API (JPA) provider implementing corresponding parts of the JSR-220 specification (aka EJB3).

Current Implementation Status

3.0 is still in development, so certain parts of the spec are not yet fully supported. Our goal is to ensure the final 3.0 release fully passes the Sun Technology Compatibility Kit (TCK), and then Cayenne will be an officially certified JPA provider.

The following tasks are currently outstanding on the road toward JPA compliance.
